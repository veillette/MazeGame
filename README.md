# Maze Game

A SceneryStack port of the PhET Maze Game simulation. Drive a particle through
mazes using Position / Velocity / Acceleration control modes — drag the arrow
in the control pad, or use arrow keys / WASD.

## Quick Start

```bash
npm install
npm run icons    # regenerate PNG icons from icons/icon.svg
npm start        # dev server → http://localhost:5173
```

## Scripts

| Command | Description |
|---|---|
| `npm start` / `npm run dev` | Start Vite dev server |
| `npm run build` | Type-check + production build → `dist/` |
| `npm run preview` | Preview the production build locally |
| `npm run check` | TypeScript type check only |
| `npm run lint` | Biome lint check |
| `npm run fix` | Lint + auto-fix |
| `npm run icons` | Regenerate PNG icons from `icons/icon.svg` |
